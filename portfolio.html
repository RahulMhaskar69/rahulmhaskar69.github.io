Academic Projects

Arduino-Based Traffic System Management

Led a team of four to design and implement an Arduino-based traffic management system. The system uses four infrared (IR) sensors to detect traffic density and dynamically control LED traffic signals.

Sensor data is processed using an Arduino Uno microcontroller. The embedded code was developed using Arduino IDE, and circuit schematics were designed using Proteus software.

A physical working model was constructed to demonstrate real-world functionality and intelligent traffic control.

Hospital ERP System

Designed the complete architecture of a Hospital ERP System using Java and Spring Boot principles. The system follows a layered architecture consisting of Controller, Service, Repository, and Database layers.

Key modules include Patient Management, Doctor & Staff Management, Appointment Scheduling, Pharmacy, Laboratory, Billing & Finance, Inventory Management, and User Security using role-based access control and JWT authentication.

The project emphasizes object-oriented design, RESTful APIs, database relationships using MySQL, and industry-standard design patterns such as MVC and Repository Pattern.
